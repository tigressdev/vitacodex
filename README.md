# Vitacodex demoapp

Estrutura pronta para deploy no Vercel com rotas:

- /demoapp/home
- /demoapp/dna-explorer
- /demoapp/clinical-sync
- /demoapp/target

## Arquivos
- `index.html` redireciona para `/demoapp/home`
- `demoapp/home/index.html`
- `demoapp/dna-explorer/index.html`
- `demoapp/clinical-sync/index.html`
- `demoapp/target/index.html`

## Deploy
1. Suba tudo para a raiz do repositório GitHub.
2. Importe no Vercel como `Other`.
3. Faça o deploy.

## Observação
As páginas usam links absolutos para funcionar em `/demoapp/...`.
